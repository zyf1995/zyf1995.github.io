使用Github Action + Hugo 实现自动化部署博客

前言 很早就想部署一个自己的博客，闲暇时间记载一下所学所思，了解到hugo能快速部署一个博客，便根据文档一步一步来操作，现在记录一下！ 一、快速开始博客搭建 step1: 下载hugo 这里介绍windows中操作： 下载链接 下载zip文件，解压到指定目录，比如D:\Software\hugo_0.82.0 将以上解压目录添加至环境变量中 在命令行工具中输入hugo version有版本信息输入说明操作成功 step2: 建立新网站 hugo new site myBlog step3: 添加主题 主题地址 cd my Blog git init git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke //将主题添加到站点配置中 echo theme = \"ananke\" >> config.toml step4: 添加一个博客 hugo new posts/my-first-post.
